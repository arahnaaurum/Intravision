Стэк: React, Redux (Redux toolkit), TypeScript, Styled Components
API: http://intravision-task.test01.intravision.ru/api/

Реализовано:
1. Стартовая страница со списком заявок
2. Форма создания заявок (открывается по клику на кнопку "Создать заявку")
3. Форма редактирования заявок (открывается при создании заявки либо по клику на карточку заявки) с возможностью:
- Добавить комментарий;
- Изменить статус;
- Изменить исполнителя.
4. Боковое меню и иные страницы приложения (содержат только заголовки, согласно ТЗ).

Примечания:
В форме редактирования при клике на поля статус / исполнитель появляется выпадающее меню.
В нем можно выбрать опцию. Меню закрывается либо при выборе опции, либо при повторном клике на поле.
